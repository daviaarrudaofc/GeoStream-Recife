# Geo-Analytics Stack (Recife Study) 🚀

Este projeto demonstra uma stack de alta performance para análise de dados geográficos.

## 🛠️ Stacks Utilizadas
- **DuckDB**: Motor de processamento OLAP in-memory.
- **H3 (Uber)**: Sistema de indexação geoespacial hexagonal.
- **GeoPandas**: Manipulação de dados geográficos e polígonos.
- **Plotly**: Visualização interativa de dados.

## 📍 Objetivo
Mapear densidade de pontos em hexágonos de resolução 8 na região de Recife/PE, utilizando SQL para processamento pesado e GeoPandas para tratamento de geometrias.
